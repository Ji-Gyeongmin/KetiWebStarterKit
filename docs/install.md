### [Node.js](https://nodejs.org)

Node.js 설치 필요

### [Gulp](http://gulpjs.com)

프론트엔드를 위한 빌드 시스템

```sh
$ npm install -g gulp
```

### [Bower](http://bower.io/)

프론트엔드를 위한 의존성 라이브러리 관리 시스템

```sh
$ npm install -g bower
```

### [Karma](http://bower.io/)

프론트엔드를 위한 의존성 라이브러리 관리 시스템

```sh
$ npm install -g karma-cli
```

### Local dependencies

의존성 라이브러리 설치

```sh
$ npm install
```

### Front-End dependencies

프론트 엔드 의존성 라이브러리 설치

```sh
$ bower install
또는
$ gulp install
```
